## 快速开始

如果您是第一次接触 Rubick，建议先阅读[安装指南](./guide/installation.md)和[使用指南](./guide/usage.md)。

如果您是开发者，想要为 Rubick 做贡献，建议从[项目概述](./development/overview.md)开始，然后阅读[架构设计](./development/architecture.md)和[插件开发指南](./development/plugin-development.md)。

## 获取帮助

如果您在使用过程中遇到问题，可以：

1. 查看[常见问题](./guide/faq.md)
2. 在 [GitHub Issues](https://github.com/rubickCenter/rubick/issues) 提交问题
3. 加入我们的[知识星球](https://rubick.vip)获取更多支持
