# Rubick 插件市场

## 插件市场概述

Rubick 插件市场是一个集成的插件管理平台，允许用户浏览、安装、更新和管理各种功能插件。插件基于 npm 包模式，安装和卸载都非常简便。

## 访问插件市场

### 通过主界面访问

1. 呼起 Rubick（使用快捷键 `Alt+R` 或 `Option+R`）
2. 点击搜索框右侧的 Rubick 图标
3. 进入插件市场界面

### 插件市场界面

插件市场界面包含以下部分：

1. **搜索框**：搜索特定插件
2. **分类标签**：按类别浏览插件
3. **插件列表**：显示插件及其信息
4. **已安装标签**：查看已安装的插件
5. **更新标签**：查看可更新的插件

## 插件分类

### 常用分类

1. **系统工具**：系统级功能增强插件
2. **开发工具**：面向开发者的工具插件
3. **效率工具**：提高工作效率的插件
4. **娱乐休闲**：娱乐和休闲相关插件
5. **网络工具**：网络相关功能插件
6. **文件管理**：文件和文件夹管理插件

### 插件类型

1. **UI 插件**：在 Rubick 主界面中运行的插件
   - 安装后立即可用
   - 通常用于特定功能操作

2. **系统插件**：集成到 Rubick 系统中的插件
   - 需要重启 Rubick 后生效
   - 通常提供系统级功能增强

## 安装插件

### 浏览和搜索

1. 在插件市场中浏览不同分类
2. 使用搜索框搜索特定插件
3. 查看插件详情和评价

### 安装步骤

1. 选择要安装的插件
2. 点击"下载"或"安装"按钮
3. 等待安装完成
4. 在"已安装"标签页中查看已安装的插件

### 插件信息

每个插件都包含以下信息：

- **插件名称**：插件的显示名称
- **插件描述**：插件功能简介
- **版本信息**：当前版本和更新历史
- **作者信息**：插件开发者信息
- **下载量**：插件下载次数
- **评分**：用户评分和评价
- **使用说明**：插件使用方法

## 管理插件

### 查看已安装插件

1. 进入插件市场
2. 点击"已安装"标签页
3. 查看所有已安装的插件列表

### 更新插件

1. 进入插件市场
2. 点击"更新"标签页
3. 查看可更新的插件
4. 点击"更新"按钮更新插件

### 卸载插件

1. 在"已安装"标签页中找到要卸载的插件
2. 点击"卸载"按钮
3. 确认卸载操作

### 插件设置

部分插件支持自定义设置：

1. 在"已安装"标签页中找到插件
2. 点击"设置"或"配置"按钮
3. 根据插件说明进行配置

## 插件源配置

### 默认插件源

Rubick 默认使用官方插件源，包含经过审核的高质量插件。

### 自定义插件源

您可以配置自定义插件源：

1. 在 Rubick 中搜索"偏好设置"
2. 进入"插件市场"选项卡
3. 修改插件源地址

### 企业内部插件源

企业用户可以配置内部插件源：

1. 搭建内部 npm 仓库
2. 在 Rubick 中配置内部插件源地址
3. 发布企业内部插件

## 常见问题

### 插件安装失败

1. 检查网络连接
2. 确认插件源地址正确
3. 尝试切换到其他插件源
4. 检查磁盘空间是否充足

### 插件无法使用

1. 确认插件已正确安装
2. 检查插件是否需要额外配置
3. 重启 Rubick（特别是系统插件）
4. 查看插件使用说明

### 插件更新问题

1. 检查网络连接
2. 确认插件源地址正确
3. 手动检查更新
