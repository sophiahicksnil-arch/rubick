# Rubick 使用指南

## 基本操作

### 启动 Rubick

您可以通过以下方式启动 Rubick：

1. **快捷键启动**：使用 `Alt+R` (Windows/Linux) 或 `Option+R` (macOS)
2. **菜单启动**：从应用程序菜单启动 Rubick
3. **托盘图标**：点击系统托盘中的 Rubick 图标

### 基本界面

Rubick 的主界面包含以下部分：

1. **搜索框**：输入关键词搜索应用、插件或文件
2. **搜索结果**：显示匹配的搜索结果
3. **插件市场入口**：点击搜索框右侧的 Rubick 图标进入插件市场

## 搜索功能

### 应用搜索

Rubick 可以搜索并启动系统中的应用程序：

1. 呼起 Rubick
2. 在搜索框中输入应用名称或拼音
3. 选择要启动的应用并按回车

**支持的搜索方式**：
- 完整应用名称：如"微信"
- 拼音：如"weixin"
- 拼音首字母：如"wx"

### 插件搜索

安装插件后，可以通过关键词搜索并使用插件：

1. 呼起 Rubick
2. 输入插件的命令关键词
3. 选择插件功能并按回车执行

### 文件搜索

Rubick 支持搜索本地文件：

1. 呼起 Rubick
2. 输入文件名或部分文件名
3. 选择文件并按回车打开

## 插件管理

### 访问插件市场

1. 呼起 Rubick
2. 点击搜索框右侧的 Rubick 图标
3. 进入插件市场

### 安装插件

1. 在插件市场中浏览或搜索插件
2. 选择需要的插件
3. 点击"下载"按钮安装
4. 安装完成后，在"已安装"标签页中查看

### 插件类型

Rubick 支持两种类型的插件：

1. **UI 插件**：在 Rubick 主界面中运行的插件
2. **系统插件**：集成到 Rubick 系统中的插件，需要重启后生效

### 卸载插件

1. 进入插件市场的"已安装"标签页
2. 选择要卸载的插件
3. 点击"卸载"按钮

## 常用功能

### 历史记录

Rubick 会记录您最近使用的插件和应用：

1. 呼起 Rubick（不输入任何内容）
2. 查看历史记录中的项目
3. 点击或使用方向键选择

### 固定常用项

您可以将常用的插件或应用固定到搜索面板：

1. 在历史记录中右键点击项目
2. 选择"固定到搜索面板"
3. 固定的项目会显示在历史记录顶部

### 剪贴板功能

Rubick 支持剪贴板内容监听：

1. 复制内容到剪贴板
2. 呼起 Rubick，剪贴板内容会自动显示
3. 可以直接对剪贴板内容进行操作

### 截图功能

使用内置截图功能：

1. 呼起 Rubick
2. 搜索"截图"或使用截图插件
3. 执行截图功能

## 设置配置

### 访问设置

1. 在 Rubick 中搜索"偏好设置"
2. 进入设置界面

### 主要设置选项

1. **快捷键设置**：修改呼起快捷键
2. **插件市场**：配置插件源
3. **多端数据同步**：配置 WebDAV 同步
4. **外观设置**：调整主题和显示选项
5. **高级设置**：配置系统级选项

## 高级技巧

### 快捷操作

- **方向键**：在搜索结果中导航
- **回车**：执行选中的操作
- **Esc**：退出 Rubick 或当前插件
- **Tab**：在搜索结果中快速切换

### 搜索技巧

1. **模糊搜索**：输入部分关键词即可匹配
2. **拼音搜索**：支持中文拼音搜索
3. **首字母搜索**：支持拼音首字母搜索
4. **正则表达式**：部分插件支持正则表达式搜索

### 插件组合使用

许多插件可以组合使用，提高工作效率：

1. 使用翻译插件翻译文本
2. 使用计算器插件进行计算
3. 使用文件管理插件快速访问文件

## 常见问题

### Rubick 无法启动

1. 检查是否正确安装
2. 尝试重新安装
3. 查看系统兼容性

### 搜索结果不准确

1. 尝试使用更精确的关键词
2. 检查插件是否正确安装
3. 清除搜索历史记录

### 插件无法使用

1. 确认插件已正确安装
2. 检查插件是否需要额外配置
3. 重启 Rubick 或系统
