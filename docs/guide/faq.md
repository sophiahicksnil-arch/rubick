# Rubick 常见问题

## 安装和启动问题

### Q: Rubick 无法启动怎么办？

**A:** 请尝试以下解决方案：

1. **检查系统兼容性**：
   - Windows: 确保使用 Windows 10 或更高版本
   - macOS: 确保使用 macOS 10.14 或更高版本
   - Linux: 确保使用 Ubuntu 18.04 或更高版本

2. **重新安装**：
   - 完全卸载当前版本
   - 下载最新版本重新安装

3. **检查权限**：
   - Windows: 以管理员身份运行
   - macOS: 在"安全性与隐私"中允许应用运行
   - Linux: 确保有执行权限

### Q: 安装时提示"无法验证发布者"怎么办？

**A:** 这是常见的安全提示，可以安全地忽略：

1. **Windows**:
   - 右键点击安装包，选择"属性"
   - 在"安全"选项卡中点击"仍要运行"

2. **macOS**:
   - 打开"系统偏好设置" > "安全性与隐私"
   - 在"通用"选项卡中点击"仍要打开"

3. **Linux**:
   - 使用命令：`sudo chmod +x rubick-x.x.x.AppImage`

### Q: 快捷键不工作怎么办？

**A:** 请尝试以下解决方案：

1. 检查快捷键是否被其他应用占用
2. 在 Rubick 设置中修改快捷键
3. 重启 Rubick 或系统
4. 检查系统快捷键设置

## 使用问题

### Q: 搜索结果不准确怎么办？

**A:** 尝试以下方法提高搜索准确性：

1. **使用更精确的关键词**：
   - 使用完整的应用名称
   - 尝试不同的关键词组合

2. **检查插件状态**：
   - 确认相关插件已正确安装
   - 检查插件是否需要更新

3. **清除搜索历史**：
   - 在设置中清除搜索历史记录
   - 重启 Rubick

### Q: 插件无法使用怎么办？

**A:** 按以下步骤排查问题：

1. **确认插件安装状态**：
   - 在插件市场的"已安装"标签页中检查插件
   - 确认插件显示为"已安装"状态

2. **检查插件类型**：
   - UI 插件：安装后立即可用
   - 系统插件：需要重启 Rubick 后生效

3. **查看插件说明**：
   - 部分插件需要额外配置
   - 检查插件的使用说明

4. **更新插件**：
   - 检查是否有可用更新
   - 更新到最新版本

### Q: 如何重置 Rubick 设置？

**A:** 重置设置的步骤：

1. **备份重要数据**：
   - 如果使用 WebDAV 同步，先导出数据

2. **清除配置文件**：
   - Windows: `%APPDATA%/rubick`
   - macOS: `~/Library/Application Support/rubick`
   - Linux: `~/.config/rubick`

3. **重新配置**：
   - 重启 Rubick
   - 重新进行初始配置

## 插件问题

### Q: 插件安装失败怎么办？

**A:** 尝试以下解决方案：

1. **检查网络连接**：
   - 确保网络连接正常
   - 尝试切换网络环境

2. **检查插件源**：
   - 确认插件源地址正确
   - 尝试切换到官方插件源

3. **检查磁盘空间**：
   - 确保有足够的磁盘空间
   - 清理临时文件

4. **重启 Rubick**：
   - 完全退出 Rubick
   - 重新启动并尝试安装

### Q: 如何卸载插件？

**A:** 卸载插件的步骤：

1. 进入插件市场
2. 点击"已安装"标签页
3. 找到要卸载的插件
4. 点击"卸载"按钮
5. 确认卸载操作

### Q: 插件更新失败怎么办？

**A:** 尝试以下解决方案：

1. **手动检查更新**：
   - 在插件市场的"更新"标签页中手动检查
   - 点击"更新"按钮

2. **重新安装插件**：
   - 先卸载插件
   - 重新安装最新版本

3. **检查插件源**：
   - 确认插件源地址正确
   - 尝试切换到其他插件源

## 数据同步问题

### Q: WebDAV 同步失败怎么办？

**A:** 按以下步骤排查：

1. **检查服务器配置**：
   - 确认 WebDAV 服务器地址正确
   - 检查用户名和密码

2. **检查网络连接**：
   - 确保可以访问 WebDAV 服务器
   - 检查防火墙设置

3. **检查权限**：
   - 确认有足够的读写权限
   - 检查服务器配置

4. **重新配置同步**：
   - 删除当前同步配置
   - 重新配置 WebDAV 设置

### Q: 如何迁移数据到新设备？

**A:** 数据迁移步骤：

1. **导出数据**：
   - 在旧设备上进入"多端数据同步"
   - 点击"导出"按钮

2. **在新设备上导入**：
   - 安装并配置 Rubick
   - 进入"多端数据同步"
   - 点击"导入"按钮

3. **验证数据**：
   - 检查插件是否正确导入
   - 确认设置已同步

## 性能问题

### Q: Rubick 运行缓慢怎么办？

**A:** 尝试以下优化方法：

1. **清理缓存**：
   - 在设置中清理缓存
   - 重启 Rubick

2. **减少启动插件**：
   - 禁用不必要的启动插件
   - 只保留常用插件

3. **更新到最新版本**：
   - 检查是否有可用更新
   - 更新到最新版本

4. **检查系统资源**：
   - 确保有足够的内存和磁盘空间
   - 关闭不必要的应用程序

### Q: 内存占用过高怎么办？

**A:** 减少内存占用的方法：

1. **清理插件缓存**：
   - 定期清理不常用的插件
   - 清理插件缓存数据

2. **限制启动插件**：
   - 只启用必要的启动插件
   - 在设置中配置启动插件

3. **重启 Rubick**：
   - 定期重启 Rubick 释放内存
   - 避免长时间运行
